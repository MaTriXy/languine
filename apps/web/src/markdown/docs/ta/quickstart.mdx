# விரைவான தொடக்கம்

Languine ஐ நிறுவுவதன் மூலம் தொடங்குங்கள்.

```bash
npm install -g @languine/cli
```

அதன்பிறகு, Languine ஐ ஆரம்பிக்க கீழ்காணும் கட்டளையை இயக்குங்கள்.

```bash
languine init
```

உங்கள் திட்டத்தை அமைக்க படிகளை பின்பற்றுங்கள்.

## முதல் மொழிபெயர்ப்பு

இப்போது நீங்கள் உங்கள் குறியீட்டை மொழிபெயர்க்க தயாராக இருக்கிறீர்கள்.

உங்கள் மூல கோப்புக்கு ஒரு விசையைச் சேர்க்க ஆரம்பிக்கவும், எடுத்துக்காட்டாக நீங்கள் `en.json` ஐ உங்கள் மூல கோப்பாக பயன்படுத்த விரும்பினால், அதில் ஒரு விசையைச் சேர்க்கவும்.

```json
{
  "hello": "வணக்கம்"
}
```

அதன்பிறகு, உங்கள் குறியீட்டை மொழிபெயர்க்க கீழ்காணும் கட்டளையை இயக்குங்கள்.

```bash
languine translate
```

நாங்கள் பின்னர் நீங்கள் சேர்த்த விசையை உங்கள் இலக்கு மொழிகளில் மொழிபெயர்க்கிறோம்.
