# GitHub செயல்கள்

GitHub செயலுடன், நீங்கள் உங்கள் குறியீட்டை எளிதாக மொழிபெயர்க்கலாம், உங்கள் குறியீட்டை சேமிப்பகத்திற்கு தள்ளுவதன் மூலம், GitHub செயலானது உங்கள் புதிய விசைகளை மொழிபெயர்க்கும், அல்லது ஒரு புல் கோரிக்கையை உருவாக்கும் அல்லது உங்கள் மாற்றங்களை உறுதிப்படுத்தும்.
இது உங்களுக்கு அதிக நேரம் மற்றும் முயற்சியைச் சேமிக்கிறது, மேலும் உயர் தரமான மொழிபெயர்ப்புகளை பராமரிக்கிறது.
