# மீறல்கள்

Languine உங்களுக்கு குறிப்பிட்ட விசைகளுக்கான மொழிபெயர்ப்புகளை மீற அனுமதிக்கிறது. இது நீங்கள் குறிப்பிட்ட விசைக்கான ஒரு குறிப்பிட்ட மொழிபெயர்ப்பு பயன்படுத்தப்படுவதை உறுதி செய்ய விரும்பும் போது பயனுள்ளதாக இருக்கும், இயல்பான மொழிபெயர்ப்பை பொருட்படுத்தாமல்.

## இது எப்படி செயல்படுகிறது

டாஷ்போர்டில், நீங்கள் குறிப்பிட்ட விசைகளுக்கான மீறல்களை உருவாக்கலாம். நீங்கள் குறிப்பிட்ட மொழிகளுக்கான மீறல்களையும் உருவாக்கலாம்.
