# GitHub Actions

Con l'azione di GitHub, puoi facilmente tradurre il tuo codice semplicemente inviando il tuo codice al repository; l'azione di GitHub tradurrà quindi le tue nuove chiavi, creando una pull request o impegnando le tue modifiche.
Questo ti fa risparmiare molto tempo e fatica, mantenendo traduzioni di alta qualità.
