# Brzi početak

Započnite instalacijom Languine-a.

```bash
npm install -g @languine/cli
```

Zatim pokrenite sljedeću naredbu za inicijalizaciju Languine-a.

```bash
languine init
```

Slijedite korake za postavljanje vašeg projekta.

## Prvi prijevod

Sada ste spremni za prevođenje vašeg koda.

Započnite dodavanjem ključa u vaš izvorni datoteku, na primjer, ako odaberete koristiti `en.json` kao vašu izvorni datoteku, dodajte ključ u nju.

```json
{
  "hello": "Zdravo"
}
```

Zatim pokrenite sljedeću naredbu za prevođenje vašeg koda.

```bash
languine translate
```

Tada ćemo prevesti ključ koji ste dodali na vaše ciljne jezike.
