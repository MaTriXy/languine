# Preklapanja

Languine vam omogućuje da preklapate prijevode za specifične ključeve. To je korisno kada želite osigurati da se određeni prijevod koristi za određeni ključ, bez obzira na zadani prijevod.

## Kako to funkcionira

Na nadzornoj ploči možete stvoriti preklapanja za specifične ključeve. Također možete stvoriti preklapanja za specifične jezike.
