# GitHub Actions

Uz github akciju, možete lako prevesti svoj kod jednostavno guranjem svog koda u repozitorij, a github akcija će zatim prevesti vaše nove ključeve, bilo da stvara pull request ili potvrđuje vaše promjene.
To vam štedi puno vremena i truda dok održavate visoku kvalitetu prijevoda.
