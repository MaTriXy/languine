# Бързо начало

Започнете с инсталирането на Languine.

```bash
npm install -g @languine/cli
```

След това изпълнете следната команда, за да инициализирате Languine.

```bash
languine init
```

Следвайте стъпките, за да настроите проекта си.

## Първичен превод

Сега сте готови да преведете кода си.

Започнете, като добавите ключ към изходния си файл, например, ако решите да използвате `en.json` като изходен файл, добавете ключ към него.

```json
{
  "hello": "Здравей"
}
```

След това изпълнете следната команда, за да преведете кода си.

```bash
languine translate
```

След това ще преведем ключа, който добавихте, на целевите ви езици.
