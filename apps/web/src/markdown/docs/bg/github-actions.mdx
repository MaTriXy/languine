# GitHub Actions

С github action можете лесно да преведете кода си, просто като го качите в репозитория. Github action след това ще преведе новите ви ключове, ще създаде pull request или ще комитне промените ви.
Това ви спестява много време и усилия, като същевременно поддържа високо качество на преводите.
