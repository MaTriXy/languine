# גיטאב אקשנס

עם פעולה של גיטאב, אתה יכול בקלות לתרגם את הקוד שלך על ידי דחיפת הקוד שלך למאגר, פעולה של גיטאב תתרגם את המפתחות החדשים שלך, או תיצור בקשת משיכה או תבצע את השינויים שלך.
זה חוסך לך הרבה זמן ומאמץ תוך שמירה על תרגומים באיכות גבוהה.
