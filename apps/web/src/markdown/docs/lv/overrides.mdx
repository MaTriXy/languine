# Pārspēki

Languine ļauj jums pārsniegt tulkojumus konkrētām atslēgām. Tas ir noderīgi, kad vēlaties nodrošināt, ka konkrēts tulkojums tiek izmantots noteiktai atslēgai, neatkarīgi no noklusējuma tulkojuma.

## Kā tas darbojas

Darbības panelī jūs varat izveidot pārspēkus konkrētām atslēgām. Jūs varat arī izveidot pārspēkus konkrētām valodām.
