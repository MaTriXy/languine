# Ātrā uzsākšana

Sāciet, instalējot Languine.

```bash
npm install -g @languine/cli
```

Pēc tam izpildiet sekojošo komandu, lai inicializētu Languine.

```bash
languine init
```

Izpildiet soļus, lai iestatītu savu projektu.

## Pirmais tulkojums

Tagad esat gatavs tulkot savu kodu.

Sāciet, pievienojot atslēgu savam avota failam, piemēram, ja izvēlaties izmantot `en.json` kā savu avota failu, pievienojiet tam atslēgu.

```json
{
  "hello": "Sveiki"
}
```

Pēc tam izpildiet sekojošo komandu, lai tulkotu savu kodu.

```bash
languine translate
```

Mēs pēc tam tulkosim atslēgu, ko pievienojāt savām mērķa valodām.
