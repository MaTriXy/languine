# GitHub Actions

Ar github darbību jūs varat viegli tulkot savu kodu, vienkārši nospiežot savu kodu uz krātuves, github darbība pēc tam tulkos jūsu jaunās atslēgas, izveidos pull pieprasījumu vai apstiprinās jūsu izmaiņas.
Tas ietaupa jums daudz laika un pūļu, saglabājot augstas kvalitātes tulkojumus.
