# GitHub Actions

S github akciou môžete jednoducho preložiť svoj kód len tým, že ho nahrajete do repozitára, github akcia potom preloží vaše nové kľúče, buď vytvorí pull request alebo uloží vaše zmeny.
To vám ušetrí veľa času a námahy, pričom zachová vysokú kvalitu prekladov.
