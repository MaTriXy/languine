# Rýchly štart

Začnite inštaláciou Languine.

```bash
npm install -g @languine/cli
```

Potom spustite nasledujúci príkaz na inicializáciu Languine.

```bash
languine init
```

Postupujte podľa krokov na nastavenie vášho projektu.

## Prvý preklad

Teraz ste pripravení preložiť svoj kód.

Začnite pridaním kľúča do vášho zdrojového súboru, napríklad ak sa rozhodnete použiť `en.json` ako váš zdrojový súbor, pridajte do neho kľúč.

```json
{
  "hello": "Ahoj"
}
```

Potom spustite nasledujúci príkaz na preloženie vášho kódu.

```bash
languine translate
```

Následne preložíme kľúč, ktorý ste pridali, do vašich cieľových jazykov.
