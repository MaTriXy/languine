# Schnellstart

Beginnen Sie mit der Installation von Languine.

```bash
npm install -g @languine/cli
```

Führen Sie dann den folgenden Befehl aus, um Languine zu initialisieren.

```bash
languine init
```

Befolgen Sie die Schritte, um Ihr Projekt einzurichten.

## Erste Übersetzung

Jetzt sind Sie bereit, Ihren Code zu übersetzen.

Beginnen Sie damit, einen Schlüssel zu Ihrer Quelldatei hinzuzufügen. Wenn Sie beispielsweise `en.json` als Ihre Quelldatei verwenden, fügen Sie einen Schlüssel hinzu.

```json
{
  "hello": "Hallo"
}
```

Führen Sie dann den folgenden Befehl aus, um Ihren Code zu übersetzen.

```bash
languine translate
```

Wir werden dann den Schlüssel, den Sie hinzugefügt haben, in Ihre Zielsprache übersetzen.
