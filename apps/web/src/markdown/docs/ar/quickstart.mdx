# البداية السريعة

ابدأ بتثبيت Languine.

```bash
npm install -g @languine/cli
```

ثم، نفذ الأمر التالي لتهيئة Languine.

```bash
languine init
```

اتبع الخطوات لإعداد مشروعك.

## الترجمة الأولى

الآن أنت جاهز لترجمة كودك.

ابدأ بإضافة مفتاح إلى ملف المصدر الخاص بك، على سبيل المثال إذا اخترت استخدام `en.json` كملف مصدر، أضف مفتاحًا إليه.

```json
{
  "hello": "مرحبا"
}
```

ثم نفذ الأمر التالي لترجمة كودك.

```bash
languine translate
```

سنقوم بعد ذلك بترجمة المفتاح الذي أضفته إلى لغاتك المستهدفة.
