# إجراءات GitHub

مع إجراء GitHub، يمكنك بسهولة ترجمة الكود الخاص بك من خلال دفع الكود إلى المستودع، ثم سيقوم إجراء GitHub بترجمة المفاتيح الجديدة الخاصة بك، إما إنشاء طلب سحب أو الالتزام بالتغييرات الخاصة بك.
هذا يوفر لك الكثير من الوقت والجهد مع الحفاظ على جودة عالية في الترجمات.
