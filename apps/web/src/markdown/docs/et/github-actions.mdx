# GitHub Actions

GitHub'i tegevuse abil saate oma koodi hõlpsasti tõlkida, lihtsalt lükates oma koodi hoidlasse. GitHub'i tegevus tõlgib seejärel teie uued võtmed, luues kas pull request'i või pannes teie muudatused toime.
See säästab teile palju aega ja vaeva, säilitades samal ajal kõrge kvaliteediga tõlked.
