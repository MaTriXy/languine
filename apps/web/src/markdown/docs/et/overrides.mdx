# Ülekirjutused

Languine võimaldab teil ülekirjutada tõlkeid konkreetsete võtmete jaoks. See on kasulik, kui soovite tagada, et konkreetset tõlget kasutatakse teatud võtme jaoks, olenemata vaike tõlkest.

## Kuidas see töötab

Armatuurlaual saate luua ülekirjutusi konkreetsete võtmete jaoks. Samuti saate luua ülekirjutusi konkreetsete keelte jaoks.
