# Hızlı Başlangıç

Languine'i kurarak başlayın.

```bash
npm install -g @languine/cli
```

Ardından, Languine'i başlatmak için aşağıdaki komutu çalıştırın.

```bash
languine init
```

Projenizi kurmak için adımları izleyin.

## İlk Çeviri

Artık kodunuzu çevirmeye hazırsınız.

Kaynak dosyanıza bir anahtar ekleyerek başlayın, örneğin `en.json` dosyasını kaynak dosya olarak kullanmayı seçerseniz, ona bir anahtar ekleyin.

```json
{
  "hello": "Merhaba"
}
```

Sonra kodunuzu çevirmek için aşağıdaki komutu çalıştırın.

```bash
languine translate
```

Eklediğiniz anahtarı hedef dillerinize çevireceğiz.
