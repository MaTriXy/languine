# GitHub Actions

GitHub eylemi ile kodunuzu sadece depoya iterek kolayca çevirebilirsiniz, GitHub eylemi yeni anahtarlarınızı çevirecek, ya bir pull request oluşturacak ya da değişikliklerinizi taahhüt edecektir.
Bu, yüksek kaliteli çevirileri korurken size çok zaman ve çaba kazandırır.
