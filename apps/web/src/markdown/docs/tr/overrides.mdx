# Geçersiz Kılmalar

Languine, belirli anahtarlar için çevirileri geçersiz kılmanıza olanak tanır. Bu, belirli bir anahtar için varsayılan çeviriden bağımsız olarak belirli bir çevirinin kullanılmasını sağlamak istediğinizde faydalıdır.

## Nasıl çalışır

Kontrol panelinde, belirli anahtarlar için geçersiz kılmalar oluşturabilirsiniz. Ayrıca, belirli diller için de geçersiz kılmalar oluşturabilirsiniz.
