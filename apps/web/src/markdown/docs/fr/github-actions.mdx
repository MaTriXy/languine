# GitHub Actions

Avec l'action GitHub, vous pouvez facilement traduire votre code en poussant simplement votre code vers le dépôt, l'action GitHub traduira alors vos nouvelles clés, soit en créant une demande de tirage, soit en validant vos modifications.
Cela vous fait gagner beaucoup de temps et d'efforts tout en maintenant des traductions de haute qualité.
