# Démarrage rapide

Commencez par installer Languine.

```bash
npm install -g @languine/cli
```

Ensuite, exécutez la commande suivante pour initialiser Languine.

```bash
languine init
```

Suivez les étapes pour configurer votre projet.

## Première traduction

Vous êtes maintenant prêt à traduire votre code.

Commencez par ajouter une clé à votre fichier source, par exemple si vous choisissez d'utiliser `en.json` comme fichier source, ajoutez-y une clé.

```json
{
  "hello": "Bonjour"
}
```

Ensuite, exécutez la commande suivante pour traduire votre code.

```bash
languine translate
```

Nous allons ensuite traduire la clé que vous avez ajoutée dans vos langues cibles.
