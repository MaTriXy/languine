# Переопределения

Languine позволяет вам переопределять переводы для конкретных ключей. Это полезно, когда вы хотите убедиться, что для определенного ключа используется конкретный перевод, независимо от перевода по умолчанию.

## Как это работает

В панели управления вы можете создавать переопределения для конкретных ключей. Вы также можете создавать переопределения для конкретных языков.
