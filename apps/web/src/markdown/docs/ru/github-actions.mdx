# GitHub Actions

С помощью действия GitHub вы можете легко переводить свой код, просто отправив его в репозиторий. Действие GitHub затем переведет ваши новые ключи, создаст запрос на слияние или зафиксирует ваши изменения.
Это сэкономит вам много времени и усилий, сохраняя при этом высокое качество переводов.
