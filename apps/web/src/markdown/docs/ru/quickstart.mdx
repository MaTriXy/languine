# Быстрый старт

Начните с установки Languine.

```bash
npm install -g @languine/cli
```

Затем выполните следующую команду для инициализации Languine.

```bash
languine init
```

Следуйте инструкциям для настройки вашего проекта.

## Первый перевод

Теперь вы готовы перевести ваш код.

Начните с добавления ключа в ваш исходный файл, например, если вы решили использовать `en.json` в качестве исходного файла, добавьте в него ключ.

```json
{
  "hello": "Привет"
}
```

Затем выполните следующую команду для перевода вашего кода.

```bash
languine translate
```

Мы затем переведем ключ, который вы добавили, на ваши целевые языки.
