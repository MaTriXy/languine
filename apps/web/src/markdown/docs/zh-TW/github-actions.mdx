# GitHub Actions

使用 GitHub Actions，您可以通過將代碼推送到存儲庫輕鬆翻譯您的代碼，GitHub Actions 將翻譯您的新鍵，並創建拉取請求或提交您的更改。
這樣可以節省大量時間和精力，同時保持高質量的翻譯。
