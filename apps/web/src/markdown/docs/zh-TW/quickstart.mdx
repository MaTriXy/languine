# 快速入門

首先安裝 Languine。

```bash
npm install -g @languine/cli
```

然後，運行以下命令來初始化 Languine。

```bash
languine init
```

按照步驟設置您的項目。

## 首次翻譯

現在您已準備好翻譯您的代碼。

首先在您的源文件中添加一個鍵，例如如果您選擇使用 `en.json` 作為源文件，則向其添加一個鍵。

```json
{
  "hello": "你好"
}
```

然後運行以下命令來翻譯您的代碼。

```bash
languine translate
```

我們將翻譯您添加到目標語言的鍵。
