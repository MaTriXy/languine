# GitHub Actions

S pomocí github akce můžete snadno překládat svůj kód jednoduše tím, že svůj kód nahrajete do repozitáře, github akce poté přeloží vaše nové klíče, buď vytvoří pull request nebo zcommitují vaše změny.
To vám ušetří spoustu času a úsilí při zachování vysoké kvality překladů.
