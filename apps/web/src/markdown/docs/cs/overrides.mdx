# Přepsání

Languine vám umožňuje přepsat překlady pro konkrétní klíče. To je užitečné, když chcete zajistit, aby byl pro konkrétní klíč použit specifický překlad, bez ohledu na výchozí překlad.

## Jak to funguje

Na řídicím panelu můžete vytvořit přepsání pro konkrétní klíče. Můžete také vytvořit přepsání pro konkrétní jazyky.
