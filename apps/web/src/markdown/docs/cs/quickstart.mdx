# Rychlý start

Začněte instalací Languine.

```bash
npm install -g @languine/cli
```

Poté spusťte následující příkaz pro inicializaci Languine.

```bash
languine init
```

Postupujte podle kroků pro nastavení vašeho projektu.

## První překlad

Nyní jste připraveni přeložit svůj kód.

Začněte přidáním klíče do vašeho zdrojového souboru, například pokud se rozhodnete použít `en.json` jako váš zdrojový soubor, přidejte do něj klíč.

```json
{
  "hello": "Ahoj"
}
```

Poté spusťte následující příkaz pro překlad vašeho kódu.

```bash
languine translate
```

Poté přeložíme klíč, který jste přidali, do vašich cílových jazyků.
