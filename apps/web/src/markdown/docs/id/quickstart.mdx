# Panduan Cepat

Mulailah dengan menginstal Languine.

```bash
npm install -g @languine/cli
```

Kemudian, jalankan perintah berikut untuk menginisialisasi Languine.

```bash
languine init
```

Ikuti langkah-langkah untuk menyiapkan proyek Anda.

## Terjemahan Pertama

Sekarang Anda siap untuk menerjemahkan kode Anda.

Mulailah dengan menambahkan sebuah kunci ke file sumber Anda, misalnya jika Anda memilih untuk menggunakan `en.json` sebagai file sumber Anda, tambahkan sebuah kunci ke dalamnya.

```json
{
  "hello": "Halo"
}
```

Kemudian jalankan perintah berikut untuk menerjemahkan kode Anda.

```bash
languine translate
```

Kami kemudian akan menerjemahkan kunci yang Anda tambahkan ke dalam bahasa target Anda.
