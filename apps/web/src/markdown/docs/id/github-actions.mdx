# GitHub Actions

Dengan tindakan github, Anda dapat dengan mudah menerjemahkan kode Anda hanya dengan mendorong kode Anda ke repositori, tindakan github kemudian akan menerjemahkan kunci baru Anda, baik membuat permintaan tarik atau mengkomit perubahan Anda.
Ini menghemat banyak waktu dan usaha Anda sambil mempertahankan kualitas terjemahan yang tinggi.
