# Penggantian

Languine memungkinkan Anda untuk mengganti terjemahan untuk kunci tertentu. Ini berguna ketika Anda ingin memastikan bahwa terjemahan tertentu digunakan untuk kunci tertentu, terlepas dari terjemahan default.

## Cara kerjanya

Di dasbor, Anda dapat membuat penggantian untuk kunci tertentu. Anda juga dapat membuat penggantian untuk bahasa tertentu.
