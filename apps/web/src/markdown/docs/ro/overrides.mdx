# Suprascrieri

Languine vă permite să suprascrieți traducerile pentru chei specifice. Acest lucru este util atunci când doriți să vă asigurați că o traducere specifică este utilizată pentru o cheie anume, indiferent de traducerea implicită.

## Cum funcționează

În tabloul de bord, puteți crea suprascrieri pentru chei specifice. De asemenea, puteți crea suprascrieri pentru limbi specifice.
