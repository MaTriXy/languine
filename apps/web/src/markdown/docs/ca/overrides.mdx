# Substitucions

Languine et permet substituir les traduccions per a claus específiques. Això és útil quan vols assegurar-te que s'utilitza una traducció específica per a una clau particular, independentment de la traducció per defecte.

## Com funciona

A la taula de comandament, pots crear substitucions per a claus específiques. També pots crear substitucions per a llengües específiques.
