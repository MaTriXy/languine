# Inici ràpid

Comença instal·lant Languine.

```bash
npm install -g @languine/cli
```

A continuació, executa la següent comanda per inicialitzar Languine.

```bash
languine init
```

Segueix els passos per configurar el teu projecte.

## Primera traducció

Ara estàs llest per traduir el teu codi.

Comença afegint una clau al teu fitxer font, per exemple, si decideixes utilitzar `en.json` com a fitxer font, afegeix una clau a ell.

```json
{
  "hello": "Hola"
}
```

A continuació, executa la següent comanda per traduir el teu codi.

```bash
languine translate
```

A continuació, traduirem la clau que has afegit a les teves llengües objectiu.
