# GitHub Actions

S pomocou github akcie môžete jednoducho preložiť svoj kód jednoducho tak, že ho nahráte do repozitára, github akcia potom preloží vaše nové kľúče, buď vytvorí pull request alebo uloží vaše zmeny.
To vám ušetrí veľa času a námahy pri zachovaní vysokej kvality prekladov.
