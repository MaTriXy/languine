# Prepisy

Languine vám umožňuje prepísať preklady pre konkrétne kľúče. To je užitočné, keď chcete zabezpečiť, aby sa pre konkrétny kľúč použil špecifický preklad, bez ohľadu na predvolený preklad.

## Ako to funguje

Na ovládacom paneli môžete vytvárať prepisy pre konkrétne kľúče. Môžete tiež vytvárať prepisy pre konkrétne jazyky.
