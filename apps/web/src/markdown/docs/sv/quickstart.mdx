# Snabbstart

Den här guiden hjälper dig att komma igång med Languine i ditt projekt.

## Installation

Först, installera Languine CLI globalt:

```bash
npm install -g @languine/cli@latest
```

## Initiera Languine

```bash
languine init
```

1. Autentisera med ditt Languine-konto
2. Välj ditt projekt
3. Välj dina filer
4. Välj dina språkinställningar
5. Välj ditt källspråk
6. Välj dina målspråk
7. Välj ditt filformat
8. Välj din filtyp
9. Välj din filkodning
10. Välj din filsökväg
