# Snabbstart

Börja med att installera Languine.

```bash
npm install -g @languine/cli
```

Kör sedan följande kommando för att initiera Languine.

```bash
languine init
```

Följ stegen för att ställa in ditt projekt.

## Första Översättning

Nu är du redo att översätta din kod.

Börja med att lägga till en nyckel i din källfil, till exempel om du väljer att använda `en.json` som din källfil, lägg till en nyckel till den.

```json
{
  "hello": "Hej"
}
```

Kör sedan följande kommando för att översätta din kod.

```bash
languine translate
```

Vi kommer sedan att översätta den nyckel du lade till till dina målspråk.
