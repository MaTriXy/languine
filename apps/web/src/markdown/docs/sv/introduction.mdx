# Introduktion

Languine är en öppen källkodsplattform för lokalisering som hjälper utvecklare att översätta sina applikationer med AI-drivna översättningar. Den erbjuder ett smidigt arbetsflöde för att hantera översättningar över flera språk samtidigt som den upprätthåller hög kvalitet genom maskininlärning.

## Nyckelfunktioner

- **AI-drivna översättningar**: Utnyttja avancerade maskininlärningsmodeller för att generera högkvalitativa översättningar över 100+ språk
- **Git-integrering**: Synka dina översättningar direkt med dina Git-förråd och automatisera uppdateringar genom CI/CD-pipeliner
- **Utvecklare-först**: Byggd med utvecklare i åtanke, med en kraftfull CLI, API och GitHub Action-integrering
- **Anpassade parsers**: Stöd för olika filformat och möjligheten att skapa anpassade parsers för dina specifika behov
- **Versionshantering**: Spåra ändringar, hantera grenar och upprätthåll översättningshistorik
- **Öppen källkod**: Helt öppen källkod och tillgänglig på GitHub, vilket möjliggör transparens och bidrag från gemenskapen

## Hur det fungerar

1. **Installation**: Installera Languine CLI och konfigurera ditt projekt med en enkel initieringsprocess
2. **Anslut**: Länka ditt förråd och ställ in ditt föredragna arbetsflöde (CLI, API eller GitHub Action)
3. **Översätt**: Skicka din källinnehåll och låt Languine hantera översättningarna till dina målspråk
4. **Synka**: Synka automatiskt översatt innehåll tillbaka till ditt förråd och distribuera med tillförsikt
