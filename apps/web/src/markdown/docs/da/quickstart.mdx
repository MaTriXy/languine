# Hurtigstart

Start med at installere Languine.

```bash
npm install -g @languine/cli
```

Kør derefter følgende kommando for at initialisere Languine.

```bash
languine init
```

Følg trinene for at opsætte dit projekt.

## Første Oversættelse

Nu er du klar til at oversætte din kode.

Start med at tilføje en nøgle til din kildefil, for eksempel hvis du vælger at bruge `en.json` som din kildefil, tilføj en nøgle til den.

```json
{
  "hello": "Hej"
}
```

Kør derefter følgende kommando for at oversætte din kode.

```bash
languine translate
```

Vi vil derefter oversætte den nøgle, du har tilføjet, til dine målsprog.
