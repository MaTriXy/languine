# Ylikirjoitukset

Languine antaa sinun ylikirjoittaa käännöksiä tietyille avaimille. Tämä on hyödyllistä, kun haluat varmistaa, että tiettyä käännöstä käytetään tietylle avaimelle, riippumatta oletuskäännöksestä.

## Kuinka se toimii

Ohjauspaneelissa voit luoda ylikirjoituksia tietyille avaimille. Voit myös luoda ylikirjoituksia tietyille kielille.
