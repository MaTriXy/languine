# Nopea aloitus

Aloita asentamalla Languine.

```bash
npm install -g @languine/cli
```

Suorita sitten seuraava komento Languinen alustamiseksi.

```bash
languine init
```

Seuraa ohjeita projektisi määrittämiseksi.

## Ensimmäinen käännös

Nyt olet valmis kääntämään koodiasi.

Aloita lisäämällä avain lähdetiedostoosi, esimerkiksi jos valitset käyttää `en.json`-tiedostoa lähdetiedostona, lisää siihen avain.

```json
{
  "hello": "Hei"
}
```

Suorita sitten seuraava komento koodisi kääntämiseksi.

```bash
languine translate
```

Käännämme sitten avaimen, jonka lisäsit kohdekielillesi.
