# GitHub Actions

Завдяки дії GitHub ви можете легко перекладати свій код, просто завантажуючи його в репозиторій. Дія GitHub потім перекладе ваші нові ключі, створить запит на злиття або зафіксує ваші зміни.
Це економить вам багато часу та зусиль, зберігаючи при цьому високу якість перекладів.
