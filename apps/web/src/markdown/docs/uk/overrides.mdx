# Перекриття

Languine дозволяє вам перекривати переклади для конкретних ключів. Це корисно, коли ви хочете забезпечити використання конкретного перекладу для певного ключа, незалежно від стандартного перекладу.

## Як це працює

У панелі керування ви можете створювати перекриття для конкретних ключів. Ви також можете створювати перекриття для конкретних мов.
