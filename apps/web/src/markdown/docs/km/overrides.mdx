# ការប្តូរ

Languine អនុញ្ញាតឱ្យអ្នកប្តូរប្រែសម្រួលសម្រាប់កូនសោជាក់លាក់។ នេះមានប្រយោជន៍នៅពេលដែលអ្នកចង់ធានាថាប្រែសម្រួលជាក់លាក់មួយត្រូវបានប្រើសម្រាប់កូនសោជាក់លាក់ណាមួយ ដោយមិនគិតពីប្រែសម្រួលលំនាំដើម។

## វិធីដែលវាធ្វើការ

នៅក្នុងផ្ទាំងគ្រប់គ្រង អ្នកអាចបង្កើតការប្តូរសម្រាប់កូនសោជាក់លាក់។ អ្នកក៏អាចបង្កើតការប្តូរសម្រាប់ភាសាជាក់លាក់ផងដែរ។
