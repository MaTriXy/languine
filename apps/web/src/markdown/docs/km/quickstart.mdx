# ការចាប់ផ្តើមយ៉ាងឆាប់រហ័ស

ចាប់ផ្តើមដោយការតំឡើង Languine។

```bash
npm install -g @languine/cli
```

បន្ទាប់មក ប្រើកំណត់ការបញ្ជា ខាងក្រោមដើម្បីចាប់ផ្តើម Languine។

```bash
languine init
```

អនុវត្តន៍ជំហានដើម្បីកំណត់គម្រោងរបស់អ្នក។

## ការប្រែសម្រួលដំបូង

ឥឡូវនេះអ្នកបានត្រៀមខ្លួនសម្រាប់ការប្រែសម្រួលកូដរបស់អ្នក។

ចាប់ផ្តើមដោយបន្ថែមកូនសោមួយទៅឯកសារដើមរបស់អ្នក ដូចជា ប្រសិនបើអ្នកជ្រើសរើសប្រើ `en.json` ជាឯកសារដើមរបស់អ្នក បន្ថែមកូនសោមួយទៅវា។

```json
{
  "hello": "សួស្តី"
}
```

បន្ទាប់មក ប្រើកំណត់ការបញ្ជា ខាងក្រោមដើម្បីប្រែសម្រួលកូដរបស់អ្នក។

```bash
languine translate
```

យើងនឹងប្រែសម្រួលកូនសោដែលអ្នកបានបន្ថែមទៅភាសាគោលដៅរបស់អ្នក។
