# GitHub Actions

Với hành động github, bạn có thể dễ dàng dịch mã của mình chỉ bằng cách đẩy mã lên kho lưu trữ, hành động github sẽ dịch các khóa mới của bạn, hoặc tạo một yêu cầu kéo hoặc cam kết các thay đổi của bạn.
Điều này giúp bạn tiết kiệm rất nhiều thời gian và công sức trong khi vẫn duy trì chất lượng dịch thuật cao.
