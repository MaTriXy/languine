# Hướng dẫn nhanh

Bắt đầu bằng cách cài đặt Languine.

```bash
npm install -g @languine/cli
```

Sau đó, chạy lệnh sau để khởi tạo Languine.

```bash
languine init
```

Thực hiện theo các bước để thiết lập dự án của bạn.

## Dịch thuật đầu tiên

Bây giờ bạn đã sẵn sàng để dịch mã của mình.

Bắt đầu bằng cách thêm một khóa vào tệp nguồn của bạn, ví dụ nếu bạn chọn sử dụng `en.json` làm tệp nguồn, hãy thêm một khóa vào nó.

```json
{
  "hello": "Xin chào"
}
```

Sau đó, chạy lệnh sau để dịch mã của bạn.

```bash
languine translate
```

Chúng tôi sẽ dịch khóa mà bạn đã thêm sang các ngôn ngữ mục tiêu.
