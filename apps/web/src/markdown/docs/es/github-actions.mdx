# GitHub Actions

Con la acción de GitHub, puedes traducir tu código fácilmente simplemente al subir tu código al repositorio; la acción de GitHub traducirá tus nuevas claves, ya sea creando una solicitud de extracción o confirmando tus cambios.
Esto te ahorra mucho tiempo y esfuerzo mientras mantienes traducciones de alta calidad.
