# GitHub Actions

ด้วย GitHub Action คุณสามารถแปลโค้ดของคุณได้อย่างง่ายดายเพียงแค่ผลักดันโค้ดของคุณไปยังที่เก็บข้อมูล GitHub Action จะทำการแปลคีย์ใหม่ของคุณ จากนั้นสร้าง pull request หรือ commit การเปลี่ยนแปลงของคุณ
สิ่งนี้ช่วยประหยัดเวลาและความพยายามของคุณในขณะที่รักษาคุณภาพการแปลที่สูง
