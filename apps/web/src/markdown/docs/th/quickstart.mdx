# เริ่มต้นอย่างรวดเร็ว

เริ่มต้นโดยการติดตั้ง Languine.

```bash
npm install -g @languine/cli
```

จากนั้น รันคำสั่งต่อไปนี้เพื่อเริ่มต้น Languine.

```bash
languine init
```

ทำตามขั้นตอนเพื่อจัดตั้งโปรเจกต์ของคุณ.

## การแปลครั้งแรก

ตอนนี้คุณพร้อมที่จะแปลโค้ดของคุณแล้ว.

เริ่มต้นโดยการเพิ่มคีย์ไปยังไฟล์ต้นฉบับของคุณ เช่น ถ้าคุณเลือกใช้ `en.json` เป็นไฟล์ต้นฉบับ ให้เพิ่มคีย์ไปยังไฟล์นั้น.

```json
{
  "hello": "สวัสดี"
}
```

จากนั้น รันคำสั่งต่อไปนี้เพื่อแปลโค้ดของคุณ.

```bash
languine translate
```

เราจะทำการแปลคีย์ที่คุณเพิ่มไปยังภาษาที่คุณต้องการ.
