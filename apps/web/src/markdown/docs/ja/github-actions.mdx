# GitHub Actions

GitHub Actionsを使用すると、リポジトリにコードをプッシュするだけで簡単にコードを翻訳できます。GitHub Actionsは新しいキーを翻訳し、プルリクエストを作成するか、変更をコミットします。
これにより、高品質な翻訳を維持しながら、多くの時間と労力を節約できます。
