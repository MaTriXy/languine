# クイックスタート

Languineをインストールすることから始めましょう。

```bash
npm install -g @languine/cli
```

次に、以下のコマンドを実行してLanguineを初期化します。

```bash
languine init
```

プロジェクトの設定手順に従ってください。

## 最初の翻訳

これでコードを翻訳する準備が整いました。

まず、ソースファイルにキーを追加します。たとえば、`en.json`をソースファイルとして使用する場合は、それにキーを追加します。

```json
{
  "hello": "こんにちは"
}
```

次に、以下のコマンドを実行してコードを翻訳します。

```bash
languine translate
```

追加したキーをターゲット言語に翻訳します。
