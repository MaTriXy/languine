# Υπερκαλύψεις

Η Languine σας επιτρέπει να υπερκαλύπτετε τις μεταφράσεις για συγκεκριμένα κλειδιά. Αυτό είναι χρήσιμο όταν θέλετε να διασφαλίσετε ότι μια συγκεκριμένη μετάφραση χρησιμοποιείται για ένα συγκεκριμένο κλειδί, ανεξαρτήτως της προεπιλεγμένης μετάφρασης.

## Πώς λειτουργεί

Στον πίνακα ελέγχου, μπορείτε να δημιουργήσετε υπερκαλύψεις για συγκεκριμένα κλειδιά. Μπορείτε επίσης να δημιουργήσετε υπερκαλύψεις για συγκεκριμένες γλώσσες.
