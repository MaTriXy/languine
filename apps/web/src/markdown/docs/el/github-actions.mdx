# GitHub Actions

Με την ενέργεια github, μπορείτε εύκολα να μεταφράσετε τον κώδικά σας απλά σπρώχνοντας τον κώδικά σας στο αποθετήριο, η ενέργεια github θα μεταφράσει τα νέα σας κλειδιά, είτε θα δημιουργήσει ένα pull request είτε θα δεσμεύσει τις αλλαγές σας.
Αυτό σας εξοικονομεί πολύ χρόνο και προσπάθεια ενώ διατηρεί υψηλής ποιότητας μεταφράσεις.
