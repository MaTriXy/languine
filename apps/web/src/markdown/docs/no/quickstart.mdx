# Hurtigstart

Start med å installere Languine.

```bash
npm install -g @languine/cli
```

Deretter, kjør følgende kommando for å initialisere Languine.

```bash
languine init
```

Følg trinnene for å sette opp prosjektet ditt.

## Første oversettelse

Nå er du klar til å oversette koden din.

Start med å legge til en nøkkel i kildefilen din, for eksempel hvis du velger å bruke `en.json` som kildefil, legg til en nøkkel i den.

```json
{
  "hello": "Hei"
}
```

Kjør deretter følgende kommando for å oversette koden din.

```bash
languine translate
```

Vi vil deretter oversette nøkkelen du la til til målspråkene dine.
