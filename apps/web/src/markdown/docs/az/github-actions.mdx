# GitHub Actions

GitHub action ilə, kodunuzu sadəcə repositoriyaya göndərməklə asanlıqla tərcümə edə bilərsiniz, GitHub action yeni açarlarını tərcümə edəcək, ya bir pull request yaradacaq, ya da dəyişikliklərinizi commitleyəcək.
Bu, sizə çox vaxt və əziyyət qazandırır, eyni zamanda yüksək keyfiyyətli tərcümələri qoruyur.
