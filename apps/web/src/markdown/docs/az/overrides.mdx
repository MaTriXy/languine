# Yeniləmələr

Languine, müəyyən açarlar üçün tərcümələri yeniləməyə imkan verir. Bu, müəyyən bir açar üçün spesifik bir tərcümənin istifadə olunmasını təmin etmək istədiyiniz zaman faydalıdır, standart tərcümədən asılı olmayaraq.

## Necə işləyir

İdarəetmə panelində, müəyyən açarlar üçün yeniləmələr yarada bilərsiniz. Həmçinin, müəyyən dillər üçün yeniləmələr də yarada bilərsiniz.
