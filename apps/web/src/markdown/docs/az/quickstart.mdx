# Tez Başlanğıc

Languine-i quraşdırmağa başlayın.

```bash
npm install -g @languine/cli
```

Sonra, Languine-i başlatmaq üçün aşağıdakı əmri icra edin.

```bash
languine init
```

Projenizi qurmaq üçün addımları izləyin.

## İlk Tərcümə

İndi kodunuzu tərcümə etməyə hazırsınız.

Mənbə faylınıza bir açar əlavə etməyə başlayın, məsələn, əgər `en.json`-u mənbə faylınız kimi istifadə etməyə qərar versəniz, ona bir açar əlavə edin.

```json
{
  "hello": "Salam"
}
```

Sonra kodunuzu tərcümə etmək üçün aşağıdakı əmri icra edin.

```bash
languine translate
```

Sonra əlavə etdiyiniz açarı hədəf dillərinizə tərcümə edəcəyik.
