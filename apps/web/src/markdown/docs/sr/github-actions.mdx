# GitHub Akcije

Sa GitHub akcijom, možete lako prevesti svoj kod jednostavno tako što ćete ga poslati u repozitorijum, a GitHub akcija će zatim prevesti vaše nove ključeve, bilo da kreira zahtev za povlačenje ili da potvrdi vaše izmene.
Ovo vam štedi mnogo vremena i truda dok održavate visoko kvalitetne prevode.
