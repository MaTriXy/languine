# Brzi početak

Započnite instalacijom Languine-a.

```bash
npm install -g @languine/cli
```

Zatim pokrenite sledeću komandu da inicijalizujete Languine.

```bash
languine init
```

Pratite korake da postavite svoj projekat.

## Prvi prevod

Sada ste spremni da prevedete svoj kod.

Započnite dodavanjem ključa u svoj izvorni fajl, na primer, ako odlučite da koristite `en.json` kao svoj izvorni fajl, dodajte ključ u njega.

```json
{
  "hello": "Zdravo"
}
```

Zatim pokrenite sledeću komandu da prevedete svoj kod.

```bash
languine translate
```

Tada ćemo prevesti ključ koji ste dodali na vaše ciljne jezike.
