# התחלה מהירה

התחל בהתקנת Languine.

```bash
npm install -g @languine/cli
```

לאחר מכן, הרץ את הפקודה הבאה כדי לאתחל את Languine.

```bash
languine init
```

עקוב אחרי הצעדים כדי להגדיר את הפרויקט שלך.

## תרגום ראשון

עכשיו אתה מוכן לתרגם את הקוד שלך.

התחל על ידי הוספת מפתח לקובץ המקור שלך, לדוגמה אם תבחר להשתמש ב- `en.json` כקובץ המקור שלך, הוסף לו מפתח.

```json
{
  "hello": "שלום"
}
```

לאחר מכן הרץ את הפקודה הבאה כדי לתרגם את הקוד שלך.

```bash
languine translate
```

לאחר מכן נתרגם את המפתח שהוספת לשפות היעד שלך.
