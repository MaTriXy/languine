# GitHub Actions

github action을 사용하면 코드를 저장소에 푸시하기만 하면 쉽게 번역할 수 있습니다. 그러면 github action이 새로운 키를 번역하고, 풀 리퀘스트를 생성하거나 변경 사항을 커밋합니다.
이것은 높은 품질의 번역을 유지하면서 많은 시간과 노력을 절약해 줍니다.
