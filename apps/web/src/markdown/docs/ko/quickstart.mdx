# 빠른 시작

Languine을 설치하는 것으로 시작하세요.

```bash
npm install -g @languine/cli
```

그런 다음, Languine을 초기화하기 위해 다음 명령어를 실행하세요.

```bash
languine init
```

프로젝트를 설정하기 위한 단계를 따르세요.

## 첫 번째 번역

이제 코드를 번역할 준비가 되었습니다.

소스 파일에 키를 추가하는 것으로 시작하세요. 예를 들어 `en.json`을 소스 파일로 선택했다면, 그 파일에 키를 추가하세요.

```json
{
  "hello": "안녕하세요"
}
```

그런 다음, 코드를 번역하기 위해 다음 명령어를 실행하세요.

```bash
languine translate
```

그러면 추가한 키를 대상 언어로 번역할 것입니다.
