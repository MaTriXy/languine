# Gyors kezdés

Kezdje a Languine telepítésével.

```bash
npm install -g @languine/cli
```

Ezután futtassa az alábbi parancsot a Languine inicializálásához.

```bash
languine init
```

Kövesse a lépéseket a projekt beállításához.

## Első fordítás

Most már készen áll a kódja fordítására.

Kezdje azzal, hogy hozzáad egy kulcsot a forrásfájlhoz, például ha az `en.json`-t választja forrásfájlul, adjon hozzá egy kulcsot.

```json
{
  "hello": "Helló"
}
```

Ezután futtassa az alábbi parancsot a kódja fordításához.

```bash
languine translate
```

Ezután lefordítjuk a hozzáadott kulcsot a célnyelvére.
