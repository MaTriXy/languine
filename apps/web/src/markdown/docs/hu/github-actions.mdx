# GitHub Actions

A GitHub akcióval könnyedén lefordíthatja a kódját, csak annyit kell tennie, hogy feltölti a kódját a tárolóba, a GitHub akció ezután lefordítja az új kulcsait, létrehoz egy pull requestet vagy elkötelezi a változtatásait.
Ez rengeteg időt és energiát takarít meg, miközben magas színvonalú fordításokat biztosít.
