# Felülírások

A Languine lehetővé teszi, hogy felülírja a fordításokat bizonyos kulcsokhoz. Ez hasznos, ha biztosítani szeretné, hogy egy adott fordítást használjanak egy bizonyos kulcshoz, függetlenül az alapértelmezett fordítástól.

## Hogyan működik

A vezérlőpulton létrehozhat felülírásokat bizonyos kulcsokhoz. Emellett létrehozhat felülírásokat bizonyos nyelvekhez is.
