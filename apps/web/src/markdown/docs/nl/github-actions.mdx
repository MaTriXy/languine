# GitHub Acties

Met de GitHub-actie kun je eenvoudig je code vertalen door je code naar de repository te pushen. De GitHub-actie vertaalt vervolgens je nieuwe sleutels, maakt een pull request of commit je wijzigingen.
Dit bespaart je veel tijd en moeite terwijl je hoogwaardige vertalingen behoudt.
