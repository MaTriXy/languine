# GitHub Actions

Com a ação do GitHub, você pode facilmente traduzir seu código apenas enviando-o para o repositório; a ação do GitHub então traduzirá suas novas chaves, criando um pull request ou confirmando suas alterações.
Isso economiza muito tempo e esforço, mantendo traduções de alta qualidade.
