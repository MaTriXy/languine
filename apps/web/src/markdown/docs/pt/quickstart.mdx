# Início Rápido

Comece instalando o Languine.

```bash
npm install -g @languine/cli
```

Em seguida, execute o seguinte comando para inicializar o Languine.

```bash
languine init
```

Siga os passos para configurar seu projeto.

## Primeira Tradução

Agora você está pronto para traduzir seu código.

Comece adicionando uma chave ao seu arquivo de origem, por exemplo, se você optar por usar `en.json` como seu arquivo de origem, adicione uma chave a ele.

```json
{
  "hello": "Olá"
}
```

Depois, execute o seguinte comando para traduzir seu código.

```bash
languine translate
```

Nós então traduziremos a chave que você adicionou para seus idiomas-alvo.
