# GitHub Actions

通过 GitHub Action，您可以轻松地通过将代码推送到代码库来翻译您的代码，GitHub Action 将翻译您的新键，或者创建一个拉取请求或提交您的更改。
这为您节省了大量时间和精力，同时保持高质量的翻译。
